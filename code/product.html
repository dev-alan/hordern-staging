Special Spirit Levels

The special spirit levels section includes products which have evolved from the tried-and-tested classic spirit levels. This has produced measuring equipment which makes work much easier and ensures reliability for very specific applications. This makes it possible to carry out the work quickly, safely and in a targeted manner. The basic products are always high-quality STABILA spirit levels, which are turned into specialised measuring equipment using specially developed additional components.
